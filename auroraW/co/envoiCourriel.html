Envoi d'un message et réception

Consignes :

  1. Dans le menu de gauche, cliquez sur "Nouveau message"

  2. Dans le champ « À », saisissez votre adresse email (oui vous pouvez vous écrire à vous même, un bon moyen de s'entraîner)

  3. Saisissez l'objet et le texte du message

  4. Cliquez sur le bouton "Envoyer"

  5. Le message devrait apparaître quelques instants plus tard dans votre boîte de réception

  6. Cliquer sur le message afin d'en voir le contenu (les messages non-lus apparaissent en gras)

  7. Répondre à ce message comme montré dans la vidéo

Sur l'image ci-contre :

A : Champ pour entrer l'adresse courriel de ou des personnes à qui vous souhaitez envoyer le courriel

B : Champs CC et CCI pour envoyer le courriel en copie carbone et copie carbone invisible

C : Outil de mise en forme

D : Pour ajouter une pièce jointe

E : Autres options pour votre message
