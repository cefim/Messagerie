Classer et organiser ses courriels avec les libellés

Pour retrouver plus facilement un mail dans votre messagerie, il faut parfois les organiser.

La plupart des messageries permettent de ranger les messages de façon automatique. Le tri peut s'effectuer par :

  • expéditeur

  • objet (sujet/titre)

  • date (le message le plus récent au-dessus)

En général, c'est ce tri qui est choisi par défaut afin de vous montrer vos nouveaux courriels en ouvrant la messagerie ; dans Gmail vous pouvez trier les courriels en créant des catégories appelés libellés (tags ou étiquettes).

En suivant la vidéo ci-dessous, créer plusieurs libellés afin de les appliquer sur vos courriels, aux besoins renvoyez-vous plusieurs courriels comme expliqué dans l'exercice précédent.

Comment gérer les libellés avec Gmail ?

Consignes :

Créer un libellé "Test formation" et y attribuer le message que vous êtes envoyé dans l'exercice précédent.

Ranger ensuite cet email dans le dossier qui a été créé automatiquement avec ce même nom de libellé.
